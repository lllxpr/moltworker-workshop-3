# Moltworker Workshop Edition

[![Deploy to Cloudflare](https://deploy.workers.cloudflare.com/button)](https://deploy.workers.cloudflare.com/?url=https://github.com/lllxpr/moltworker-workshop)

> Deploy your own AI agent on Cloudflare in minutes — powered by **Workers AI Kimi K2.5**.

This is a **workshop-ready fork** of [cloudflare/moltworker](https://github.com/cloudflare/moltworker) (OpenClaw) with an interactive setup wizard that automates the entire deployment process.

## Two Ways to Deploy

### Option 1: One-Click Deploy (Recommended for Quick Start)
Click the **"Deploy to Cloudflare"** button above. Cloudflare will build the container in the cloud — **no local Docker needed**!

After deployment, you'll still need to set secrets via the wizard:
```bash
npm run wizard
```

### Option 2: Local Wizard (Full Automation)
Clone the repo and run the wizard locally (requires Docker for container builds):
```bash
git clone https://github.com/lllxpr/moltworker-workshop.git
cd moltworker-workshop
npm install
npm run wizard
```

## What's Different from the Official Repo?

| Feature | Official | Workshop Edition |
|---------|----------|-----------------|
| Setup | Manual secret-by-secret | **One-command wizard** |
| AI Model | Anthropic Claude (requires API key) | **Workers AI Kimi K2.5** (no external key) |
| Billing | BYOK (Bring Your Own Key) | **Unified Billing** (Cloudflare bills directly) |
| AI Gateway | Optional, manual setup | **Auto-created** by wizard |
| R2 Storage | Manual bucket + token | **Auto-configured** by wizard |
| Access | Manual Cloudflare Access setup | **Guided setup** with wizard |
| Cost | Requires Anthropic credits | **Pay-as-you-go** via Cloudflare |

## Quick Start

```bash
# 1. Clone this repo
git clone https://github.com/lllxpr/moltworker-workshop.git
cd moltworker-workshop

# 2. Install dependencies
npm install

# 3. Run the setup wizard — it handles everything!
npm run wizard
```

The wizard will:
1. ✅ Verify your environment (Node.js, Wrangler, Cloudflare login)
2. ✅ Detect your Cloudflare Account ID
3. ✅ Create an AI Gateway instance
4. ✅ Generate an AI Gateway authentication token
5. ✅ Select the AI model (default: Kimi K2.5)
6. ✅ Generate a secure gateway token
7. ✅ Create R2 bucket for persistent storage
8. ✅ Create R2 API credentials
9. ✅ Set all required secrets automatically
10. ✅ Guide you through Cloudflare Access setup
11. ✅ Deploy to Cloudflare Workers

## Requirements

- **Node.js 18+** — [Download](https://nodejs.org)
- **Cloudflare Workers Paid plan** ($5/month) — required for Sandbox containers
- A Cloudflare account (workshop attendees: check your email for the invitation)

## All Secrets (Reference)

The wizard sets these automatically. For manual setup, see the table below:

| Secret | Purpose | How to Get |
|--------|---------|------------|
| `CLOUDFLARE_AI_GATEWAY_API_KEY` | AI Gateway auth token | Created in AI Gateway → Settings → Authentication |
| `CF_AI_GATEWAY_ACCOUNT_ID` | Cloudflare Account ID | Dashboard sidebar or `wrangler whoami` |
| `CF_AI_GATEWAY_GATEWAY_ID` | AI Gateway instance ID | Created by wizard or in AI Gateway dashboard |
| `CF_AI_GATEWAY_MODEL` | AI model to use | `workers-ai/@cf/moonshotai/kimi-k2.5` |
| `MOLTBOT_GATEWAY_TOKEN` | Token for Control UI access | Randomly generated by wizard |
| `R2_ACCESS_KEY_ID` | R2 API access key | R2 → Manage API Tokens |
| `R2_SECRET_ACCESS_KEY` | R2 API secret key | R2 → Manage API Tokens |
| `CF_ACCOUNT_ID` | Account ID (for R2) | Dashboard sidebar |
| `CF_ACCESS_TEAM_DOMAIN` | Zero Trust team domain | Zero Trust → Settings |
| `CF_ACCESS_AUD` | Access application AUD tag | Access → Applications → your app |
| `SANDBOX_SLEEP_AFTER` | Auto-sleep idle timeout | Set to `10m` to save costs |

## After Deployment

### Access Your Agent

```
# Control UI (chat interface)
https://<your-worker>.workers.dev/?token=YOUR_GATEWAY_TOKEN

# Admin UI (device management)
https://<your-worker>.workers.dev/_admin/
```

### Pair Your Device

1. Open the Admin UI (`/_admin/`)
2. Log in via Cloudflare Access
3. Click "Pair Device" and approve

## Cost Estimate

With `SANDBOX_SLEEP_AFTER=10m` (auto-sleep after 10 min idle):
- **Workers Paid plan**: $5/month
- **Container compute** (~4 hrs/day): ~$5-6/month
- **Workers AI (Kimi K2.5)**: Pay-per-token, very affordable
- **R2, AI Gateway, Access**: Free tier

**Total**: ~$10-11/month for casual use
